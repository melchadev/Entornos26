# Entornos26 - Base de Desarrollo

Este repositorio forma parte de las prácticas del módulo **Entornos de Desarrollo** del ciclo **Desarrollo de Aplicaciones Multiplataforma (DAM)**.

El objetivo de este repositorio es servir como base para aplicar herramientas y metodologías utilizadas en el desarrollo profesional de software.

---

## 🎯 Objetivos del proyecto

- Configuración de un entorno de desarrollo profesional con **IntelliJ IDEA**
- Uso de **Git y GitHub** para el control de versiones
- Creación y organización de un repositorio de trabajo
- Documentación técnica mediante **README**

---

## 🛠️ Herramientas y tecnologías

Las herramientas utilizadas en este proyecto son:

- **Java**
- **IntelliJ IDEA**
- **Git**
- **GitHub**
